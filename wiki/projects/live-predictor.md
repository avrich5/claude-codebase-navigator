# Live Predictor

Сервис для работы с API Analysator в режиме реального времени. Получает данные с Binance, отправляет их предиктору и анализирует точность предсказаний.

## Возможности

- Получение данных с публичного API Binance
- Калибровка предиктора на исторических данных
- Получение и валидация предсказаний в режиме реального времени
- Визуализация результатов и метрик точности
- Отказоустойчивость и автоматическое восстановление

## Установка

```
pip install -r requirements.txt
```

## Общий обзор проекта

Ваш проект "Live Predictor" получает данные с Binance, отправляет их в API Analysator, и анализирует точность предсказаний. Вы хотите переключиться на отслеживание ETHUSDT вместо BTCUSDC.

## Порядок действий для запуска

1. **Изменить конфигурацию**:
   - Изменить торговую пару на ETHUSDT
   - Отключить гибридный предиктор

2. **Проверить API и выполнить сброс**:
   - Сначала нужно проверить доступность API
   - Затем сбросить предиктор перед использованием

3. **Запустить основной сервис**:
   - Запустить сервис, который будет получать данные и отправлять их предиктору

4. **Мониторинг**:
   - Использовать встроенный монитор для отслеживания метрик
