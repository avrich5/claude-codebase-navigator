# Strategy Market - Design Concept

**Marketing Landing Page для токенизированного маркетплейса торговых стратегий**

##  Дизайн-концепция

Этот проект - **маркетинговая** версия платформы, созданная для презентации инвесторам и партнерам. В отличие от production сайта (profit-radar-io), здесь фокус на продажах, эмоциях и конверсии.

### Вдохновение
- **UI стиль:** [Huly.io](https://huly.io) - темная тема, контрасты, микроанимации
- **Маркетинг:** [Wonderful.ai](https://www.wonderful.ai) - эмоциональный storytelling
- **Концепция:** PumpFun механика для algo trading стратегий

### 4 Ключевых принципа дизайна

#### 1. Визуальная иерархия через контраст
- Темный фон (`#0a0a0f`) + яркие акценты (`#00ff88`, `#6366f1`)
- Крупные заголовки (4rem) с gradient эффектами
- Четкое разделение секций

#### 2. Микроанимации и движение
- Particle system на фоне
- Floating cards (поднимаются вверх-вниз)
- Counter animations (числа анимируются)
- Typing effects (текст печатается)
- Countdown timers (живой отсчет)

#### 3. Типографика
- Font: System fonts (-apple-system, Segoe UI, Roboto)
- Крупные заголовки: 3-4rem, bold, tight line-height
- Body text: 1.25rem, secondary color
- Gradient text для акцентов

#### 4. Продуктовые screenshots/mockups
- Реальные чарты из profit-radar-ai
- Live performance cards
- Creator earnings dashboard
- Token lifecycle visualization
