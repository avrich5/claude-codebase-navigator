## 📝 README

```
# Гибридный предиктор временных рядов

Этот проект реализует гибридный предиктор временных рядов, объединяющий марковские цепи и квантильную регрессию для прогнозирования движения цен на финансовых рынках.

## Особенности проекта

- Гибридный подход, сочетающий марковские цепи для определения состояния рынка и квантильную регрессию для оценки будущих значений
- Расчёт уверенности в предсказаниях с поддержкой адаптивных порогов
- Оптимизированные метрики для трейдинга (precision, confusion matrix)
- Поддержка различных режимов работы и конфигураций
- Инструменты для валидации и визуализации результатов

## Структура проекта

```
.
├── config.py                    # Конфигурации и параметры предиктора
├── predictor_mq/                # Основной код предиктора
│   └── models/
│       ├── hybrid_predictor.py  # Реализация гибридного предиктора
│       └── standardized_quantile_regression.py  # Модель квантильной регрессии
│
├── data/                        # Данные для обучения и тестирования
│   ├── train/                   # Тренировочные данные
│   ├── validation/              # Данные для валидации
│   └── market_mode_data/        # Данные по разным режимам рынка
│
├── hybrid_model_api/            # API для внешнего использования модели
│   ├── model_api.py             # Основной API интерфейс
│   ├── train_model.py           # Скрипт для обучения моделей
│   └── saved_models/            # Сохраненные обученные модели
│
├── examples/                    # Примеры использования
```
